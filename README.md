# Flipkart_Webscrapping

**Scrapped the data of the Macbook's available on Flipkart.com**

Libraries used :
 * BeautifulSoup
 * Time
 * Pandas
 * Requests
 
 
 **Data Collected:**
 * Product name
 * Price
 * Stars rated
 * Specifications
 * Ratings
 * Reviews
 

**Data-source link:-**
